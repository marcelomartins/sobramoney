# sobramoney

## Instação do Vue
* yarn global add @vue/cli
* yarn global add @vue/cli-service-global
* npm install

## Desenvolvimento
* yarn build --dest='../src/main/webapp/' --mode='development' --watch (no diretório vue-app)
* grails run-app


## Teste
* yarn build --dest='../src/main/webapp/' (no diretório vue-app)
* grails run-app


## Gerar Pacote
* yarn build --dest='../src/main/webapp/' (no diretório vue-app)
* grails war
